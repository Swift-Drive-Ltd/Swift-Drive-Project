# InstructorAvailabilityTracker Class
- Uses a dictionary to track availability dates for each instructor
- Methods:
  - `AddAvailability`: Adds a date to an instructor's availability
  - `IsAvailable`: Checks if an instructor is available on a specific date
  - `RemoveAvailability`: Removes a date from an instructor's availability
